# 一、加粗(Ctrl + B)，斜体(Ctrl + I)，高亮

```bash
1.加粗创建方式：
**加粗内容**
__加粗内容__

2.斜体创建内容
*斜体内容*
_斜体内容_

3.高亮创建方式：
==高亮内容==
```

**Typora**是一款支持实时预览的*Markdown*==文本编辑器==



# 二、下划线(Ctrl + U)，删除线 (Alt + Shift + 5)，清除格式(Ctrl + \)

```bash
1.下划线创建方式：
<u>文本内容</u>

2.删除线创建方式：
~~文本内容~~
```

<u>Typora</u>是一款~~编辑器~~文本编辑器



==注：可叠加效果==
