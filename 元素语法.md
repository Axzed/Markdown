# 一、图片

快捷键：Ctrl + Shift + I（image）

```bash
创建方式：![]() -> 选择图片
相关设置：文件 —> 偏好设置 —> 选择【复制图片到./$(filename).assets文件夹】 —> 勾选【优先使用相对路径】—> 重启Typora 
```

# 二、超链接

快捷键：Ctrl + k(快)

```bash
创建方式：
①直接粘贴链接
②[显示内容](网址)

访问方式：
Ctrl + 左键单击
```
